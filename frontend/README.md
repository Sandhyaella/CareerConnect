# CareerConnect - Full Stack MERN Job Portal

CareerConnect is a production-ready MERN job portal with role-based access for candidates and recruiters.

## Project Structure

- `frontend`: React + Vite app
- `backend`: Node.js + Express + MongoDB API (MVC)

## Run Locally

### Backend

1. `cd backend`
2. `npm install`
3. Copy `.env.example` to `.env` and set values
4. `npm run seed` (optional sample data)
5. `npm run dev`

### Frontend

1. `cd frontend`
2. `npm install`
3. Copy `.env.example` to `.env`
4. `npm run dev`
